📋 معلومات المشروع:
==================

اسم المشروع: نظام إدارة مهام مركز الموهوبين بحائل
المطور: الأستاذ متعب بن مطير المتعب
تاريخ الإنشاء: 2025
الإصدار: 1.0
اللغة: Python (Flask)
قاعدة البيانات: SQLite
الواجهة: HTML/CSS/JavaScript (Bootstrap)

🎯 الهدف من النظام:
==================

تطوير نظام شامل لإدارة وتوزيع المهام في مركز الموهوبين بحائل
يهدف إلى تحسين الكفاءة وتنظيم العمل بين جميع الوحدات

✨ المميزات الرئيسية:
====================

✅ إدارة المستخدمين والصلاحيات
✅ إنشاء وتوزيع المهام
✅ متابعة حالة المهام (جديد، قيد التنفيذ، مكتمل، متأخر)
✅ تحديد الأولويات (عالي، متوسط، منخفض)
✅ تحديد المواعيد النهائية
✅ تقارير شاملة وإحصائيات
✅ رسوم بيانية تفاعلية
✅ واجهة عربية سهلة الاستخدام
✅ تنبيهات المهام المتأخرة
✅ تحديث تلقائي للبيانات
✅ تصدير التقارير

👥 المسميات الوظيفية:
=====================

• مدير: جميع الصلاحيات
• وكيل: إنشاء المهام والتقارير
• معلم موهوبين: تنفيذ المهام المعينة
• إداري: تنفيذ المهام الإدارية

🏢 وحدات العمل:
================

• وحدة الإدارة
• وحدة العلاقات العامة والإعلام
• وحدة البرامج الإثرائية
• وحدة المسابقات
• وحدة شؤون الطلاب

🛠️ التقنيات المستخدمة:
======================

Backend:
- Python 3.12
- Flask 2.3.3
- SQLAlchemy (ORM)
- SQLite (قاعدة البيانات)

Frontend:
- HTML5
- CSS3
- JavaScript
- Bootstrap 5.3
- Font Awesome (الأيقونات)
- Chart.js (الرسوم البيانية)

🚀 التشغيل:
============

1. تثبيت المتطلبات: setup.bat
2. تشغيل النظام: start.bat
3. فتح المتصفح: http://localhost:5000
4. تسجيل الدخول: admin / admin123
